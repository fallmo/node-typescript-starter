# node-typescript-start

Basic Express Server with Typescript

## Scripts

- _start_: run compiled app '/dist/app.js'
- _dev_: run dev server with nodemon
- _build_: compile typescript app 'src/app.ts' to 'dist' folder

## Usage

1. Clone repo
2. Edit src/app.ts
3. Run Build Command

## devDependancies

- @types/express
- @types/node
- nodemon
- ts-node
- typescript
